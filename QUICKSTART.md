# Quick Start Guide

## Run the App

```bash
# From the discord-app directory
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Login Credentials

You can use your Matrix credentials:

**Homeserver**: `https://matrix.org`
**Username**: `mhmadnab` (just the username, not @mhmadnab:matrix.org)
**Password**: Your Matrix password

## What You'll See

1. **Login Page** - Beautiful gradient login screen
2. **Chat Interface** with 3 panels:
   - **Left Sidebar**: Your rooms/channels with + button to create new ones
   - **Center**: Message timeline with input
   - **Right Sidebar**: Member list

## Features

✅ Real-time messaging
✅ Multiple rooms (channels)
✅ Create new rooms
✅ Member list
✅ Message history
✅ Auto-reconnect
✅ Unread message counts

## Try It Out

1. Login with your credentials
2. Click the + button to create a room
3. Send messages
4. Open [Element](https://app.element.io) in another tab
5. Watch messages sync in real-time!

## Development

- **Hot Reload**: Changes update automatically
- **TypeScript**: Full type safety
- **Tailwind CSS**: Easy styling
- **Matrix SDK**: Full Matrix protocol support
